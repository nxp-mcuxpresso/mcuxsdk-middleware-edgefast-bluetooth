# Hardware rework

1.  HCI UART rework
    -   Solder R93 and R96
2.  PCM interface rework
    -   Solder R70 and R79; remove R76 and R86; Connect J80.

**Note:** Make sure to disconnect J80 when debugging. Otherwise, the debugger downloading fails.
